# FinSight Finance Analytics Dashboard

## Overview

FinSight is an interactive Finance Analytics Dashboard built in Power BI to provide comprehensive insights into financial transactions, customer behavior, fees, taxes, and overall business performance. The dashboard enables stakeholders to monitor key financial metrics, analyze transaction trends, and make data-driven business decisions through intuitive visualizations and interactive reporting.

## Business Objective

The goal of FinSight is to provide a centralized analytical platform that helps organizations:

* Monitor financial KPIs in real time
* Analyze transaction growth and performance trends
* Evaluate customer segment contributions
* Compare regional financial performance
* Track transaction success rates
* Assess fees and tax generation
* Understand customer demographics
* Support strategic business decision-making

## Key Performance Indicators (KPIs)

* Total Transaction Amount
* Total Transactions
* Average Transaction Value
* Total Fees Collected
* Total Tax Generated
* Year-over-Year (YoY) Growth Analysis

## Dashboard Features

### Financial Performance Analysis

* Monthly transaction amount trends
* YoY performance comparison
* Financial growth monitoring

### Transaction Analysis

* Success, Failed, and Pending transaction analysis
* Transaction volume tracking
* Transaction type performance evaluation

### Customer Insights

* Customer segment contribution analysis
* Gender-based transaction analysis
* Customer behavior insights

### Regional Analysis

* State-wise financial performance
* Geographic comparison of transaction activity

### Detailed Reporting

* Interactive drill-through functionality
* Detailed transaction records
* Dynamic filtering capabilities
